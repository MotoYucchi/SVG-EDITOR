SVG EDITORを起動するために、ローカルストレージへのアクセス権限が必要です。

SVG EDITOR POWERED BY MOTOYUCCHI について

この「SVG EDITOR」は、開発者 MotoYucchi (もとゆっち / モトユッチ) が現在開発を行っている、ブラウザベースの簡易ベクター画像編集ツールです。 Rect, Path, Textなどの基本要素の配置や、レイヤーのドラッグ＆ドロップによる入れ替えに対応しています。 また、GUIだけでは難しい細かい調整のために、SVGコードを直接編集できる機能(Raw Code Editor)を搭載しており、 コードレベルでの柔軟な編集が可能です。開発中のツールのため、シンプルかつ軽量な動作を目指しています。
